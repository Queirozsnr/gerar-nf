# Cypress Automation for Gerar NF

Este projeto utiliza o framework de teste end-to-end Cypress para automatizar o processo de geração de Notas Fiscais (NF). O objetivo é facilitar e acelerar os testes relacionados à geração de NF, proporcionando confiança e eficiência no processo.

## Pré-requisitos

- Certifique-se de que o [Node.js](https://nodejs.org/) está instalado em sua máquina.
- Instale as dependências do projeto utilizando o comando:

  ```bash
  npm install

## Iniciando o Projeto
- Antes de iniciar o projeto, você precisa informar seus dados no arquivo cypress/e2e/gerar-nf.cy.js
- Para rodar o projeto voce precisa utilizar o seguinte comando:
  
  ```bash
  npx cypress open
